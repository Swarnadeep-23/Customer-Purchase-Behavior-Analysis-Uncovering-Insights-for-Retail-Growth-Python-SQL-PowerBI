# Customer-Purchase-Behavior-Analysis-Uncovering-Insights-for-Retail-Growth-Python-SQL-PowerBI
Analyzed 3,900 retail transactions using Python, SQL, and Power BI to uncover customer insights, identify behavioral patterns, and inform business decisions on marketing, customer retention, and product optimization.

 Project Overview
This project focuses on analyzing customer purchase behavior using 3,900 retail transactions to uncover spending patterns, product preferences, demographic trends, and subscription behavior. The insights aim to support strategic decisions across marketing, customer engagement, and product optimization. The project covers the full analytics pipeline  from data cleaning and feature engineering in Python to SQL-based business analysis and final visualization in Power BI.

Project Duration
1 Week

Role
Data Analyst 

Key Highlights 
1. End-to-End Data Pipeline
Executed the entire analytics workflow  from raw data cleaning to SQL insights and dashboard visualization.
2. Customer Segmentation Model
Classified customers into New, Returning, and Loyal groups to identify retention and engagement opportunities.
3. Insightful Behavioral Patterns
Uncovered key drivers such as discount usage, subscription behavior, age group spending patterns, and shipping preferences.
4. Business-Ready SQL Insights
Created 10+ SQL analyses addressing real retail business questions, including revenue trends, repeat buyers, and top product categories.
5. Power BI Decision-Making Dashboard
Built an interactive dashboard enabling stakeholders to explore trends and make informed decisions backed by data.
